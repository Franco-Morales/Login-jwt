# Frontend Auth JWT

Ejemplo de frontend utilizando `Json Web Token` como método de autenticación.


## Dependencias
- **Angular** : ~13
- **RxJs**: ~7
- **Bootstrap**
- **Ngx-Toastr**
- **@Auth0/angular-jwt**
