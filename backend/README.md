# Backend Auth JWT

Ejemplo de backend utilizando `Json Web Token` como método de autenticación. Como base de datos se utiliza MongoDB Atlas.

## Dependencias
- **Express.js**
- **Mongoose** 
- **Bcrypt** 
- **Cors** 
- **Helmet** 
- **Dotenv** 
- **Express-validator** 
- **JsonWebToken** 

## Variables de entorno

El proyecto utiliza `.env` para guardar 'informacion sensible'.
